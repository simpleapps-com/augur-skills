---
description: Token budget enforcement for skills
globs: "**/SKILL.md"
---

All SKILL.md files MUST be under 5K tokens. Before finishing any skill edit, check with `wc -w` — rough conversion: 1 token ≈ 0.75 words, so 5K tokens ≈ 3,750 words max. If a skill exceeds this, consolidate and tighten before committing.
