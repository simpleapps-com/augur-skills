# SKILL.md Format

Full docs: [Skill Format](../../../wiki/Skill-Format.md)

Required frontmatter: `name` (kebab-case, max 64 chars), `description`.
Version is tracked at the plugin level (`plugin.json`), NOT in individual SKILL.md files.
